# AI Response Evaluator

In this project, I explored how AI-generated responses can be evaluated in a simple and structured way.

I worked with short user queries and compared two possible responses (A and B), selecting the best one and explaining why.

## What I did

- Compared different responses to the same question  
- Evaluated them using clear criteria  
- Chose the best option (A or B)  
- Wrote short justifications for each decision  
- Organized everything into a small dataset  

## Evaluation criteria

I focused on:

- Relevance (does it address the user's request?)  
- Clarity (is it easy to understand?)  
- Completeness (does it fully answer the question?)  
- Factuality (is the information correct?)  

## Why this matters

This kind of evaluation is part of how AI systems are improved in real-world scenarios.  
It’s especially relevant for tasks like response evaluation, data annotation, and quality control.
